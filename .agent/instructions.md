# Agent 15 - Minecraft Builder Standards

## Visual Identity (Gold Standard)
- **Style:** Medieval Engineering / Renaissance Sketch.
- **Aesthetic:** Hand-drawn pencil sketches on aged parchment (sepia tones).
- **Format:** Isometric component views with technical annotations.
- **CRITICAL:** Must look like **MINECRAFT BLOCKS**. Every element must be cubic/voxel (1x1 square blocks). Do not draw realistic pillars; draw cubic block pillars. Visible block-grid lines are required.

## Minecraft Version: 1.8.8 (Strict)
All builds and blueprints MUST use blocks available in Minecraft 1.8.8.

### Core Palette:
- **Stone:** Stone Bricks, Cobblestone, Mossy Stone Bricks.
- **Wood:** Dark Oak, Spruce, Oak (Logs and Planks).
- **Lighting:** Glowstone, Sea Lanterns, Torches.
- **Details:** Iron Bars, Nether Brick Fences, Stone Slabs/Stairs.

### Banned Materials (Post 1.8):
- NO Concrete, Terracotta, Blackstone, Deepslate, or Netherite.

## Prompting Rules for Visuals:
- Use keywords: "Vintage engineering sketch", "aged parchment", "technical annotations", "isometric view", "pencil drawn", "Minecraft 1.8.8 materials".
- Focus on detail over scale: preference for component blueprints (Wall sections, Pillars, Gates).
